아날로그 vs 디지털

아날로그 (Analog)

  • 연속적인 값의 변화
  • 예: 수은 온도계, LP판, 시침이 있는 시계
  • 단점: 보관이나 복사 시 잡음(노이즈) 발생

디지털 (Digital)

  • 끊어지는 값 (이산적인 값)
  • 예: 디지털 온도계, CD, 숫자 시계
  • 장점: 복사가 정확하고 가공이 쉬움

십진수 → 이진수 변환법

방법 1: 2로 계속 나누기

나머지 값을 거꾸로 읽으면 이진수가 됩니다.

13 / 2 = 6 ... 1
6 / 2 = 3 ... 0
3 / 2 = 1 ... 1
1 / 2 = 0 ... 1
=> 1101₂

방법 2: 자릿값 덧셈

필요한 자릿값을 더해 숫자를 만듭니다.

13 = 8 + 4 + 1
(8) (4) (2) (1)
1 1 0 1
=> 1101₂
