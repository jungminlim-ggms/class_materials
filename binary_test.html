📌 오늘의 학습 목표

  • 실생활의 데이터가 디지털 형태(이진수)로 표현되는 원리를 이해한다.
  • 십진수를 이진수로, 이진수를 십진수로 자유롭게 변환할 수 있다.
  • 게임을 통해 20개의 미션을 완수하고 이진수 마스터 배지를 획득한다.
[시각자료 안내: 컴퓨터 내부의 0과 1 신호 흐름도 배치 구역]

활동 1. 왜 컴퓨터는 이진수를 쓸까?

💡 십진수 (Decimal)

  • 사람이 일상에서 사용하는 진법
  • 0부터 9까지 총 10개의 숫자를 사용
  • 자릿수가 올라갈 때마다 10배씩 커짐 (1, 10, 100, 1000...)

💻 이진수 (Binary)

  • 컴퓨터가 디지털 데이터를 표현하는 방식
  • 0과 1 딱 두 개의 숫자만 사용
  • 전기 신호가 꺼짐(0)과 켜짐(1) 상태로 완벽 매칭
  • 자릿수가 올라갈 때마다 2배씩 커짐 (1, 2, 4, 8, 16...)
📢 핵심: 컴퓨터는 미세한 전압 변화의 오류를 줄이기 위해 가장 확실한 두 가지 상태(0, 1)만 채택했습니다!

활동 2. 십진수를 이진수로 바꾸는 마법

십진수를 더 이상 나눠지지 않을 때까지 2로 계속 나누고, 나머지를 거꾸로 읽어줍니다.

📝 예제: 십진수 13을 이진수로 바꾸기

2 ) 13 ··· 나머지 1 (우측 끝자리)

2 ) 6 ··· 나머지 0

2 ) 3 ··· 나머지 1

     1 (가장 앞자리)

➔

아래에서 위로 거꾸로 정렬

1101(2)

[시각자료 안내: 13을 2로 나누어 나눗셈 몫과 나머지가 남는 과정을 시각화한 수식 다이어그램 란]

활동 3. 이진수를 십진수로 복원하기

이진수의 각 자릿수는 뒤에서부터 1, 2, 4, 8, 16...의 고유 가중치(자리값)를 가집니다.

📝 예제: 이진수 1010을 십진수로 바꾸기

자릿수 위치 4번째 (8의 자리) 3번째 (4의 자리) 2번째 (2의 자리) 1번째 (1의 자리)
이진수 값 1 0 1 0
계산식 8 × 1 = 8 4 × 0 = 0 2 × 1 = 2 1 × 0 = 0
합산 결과: 8 + 0 + 2 + 0 = 10 입니다!
